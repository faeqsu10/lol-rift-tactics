# Tactics Milestones

## 기준

이 문서는 `native_tactics`를 어디까지 어떤 순서로 밀어야 `1차 완성`에 도달하는지 정리한다.

## M1. 전술 알파 완성

### 목표

- 지금 있는 전술 프로토타입을 "재밌는지 검증 가능한 알파"까지 끌어올린다.

### 포함 항목

- 챔피언 고유 패시브 2차
- 챔피언별 스킬 차별화 2차
- 적 의도 3차:
  - 다중 타겟
  - 연쇄 피해
  - sequence preview
- 경로 목표 실패 페널티
- AI의 목표 경쟁, 엘리트 보호, 위험 타일 우선도 개선

### 완료 기준

- 각 챔피언의 사용감이 분명히 달라진다.
- 적 턴이 억울하지 않고 읽을 수 있다.
- 맵 목표를 무시할지 노릴지 판단이 실제 전략이 된다.

## M2. 런 구조 베타

### 목표

- 한 런 전체가 지루하지 않고, 판마다 다른 선택이 나오게 만든다.

### 포함 항목

- 경로별 전용 이벤트 2차
- 엘리트/휴식/보상/이벤트 노드 분기
- 보스전 1차
- 런 종료 요약
- 런 전용 강화 다양화

### 완료 기준

- 전투만 다른 게 아니라 노드 선택 자체가 재미가 된다.
- 보스전이 일반 전투와 분명히 다르다.
- 실패 뒤 재시작 동기가 생긴다.

## M3. 콘텐츠 확장 베타

### 목표

- "재미 구조" 위에 실제 반복 플레이 가능한 볼륨을 얹는다.

### 포함 항목

- 플레이어 챔피언 8~12명 의미 있게 차별화
- 적 챔피언 8~12명 의미 있게 차별화
- 맵 타입 4종 이상
- 보스 2종 이상
- 이벤트 10종 안팎
- 목표 타입 6종 이상

### 완료 기준

- 같은 조합만 반복하지 않게 된다.
- 맵과 경로에 따라 플레이 감각이 달라진다.

## M4. 프레젠테이션/출시 후보

### 목표

- "재밌는 프로토타입"을 "배포 가능한 게임"으로 정리한다.

### 포함 항목

- 스프라이트/애니메이션 1차
- 전투 연출과 승패 연출 강화
- 배경 맵 아트
- UI 폴리시
- 설정 메뉴
- 윈도우 패키징 정리
- 외부 플레이테스트 반영

### 완료 기준

- 외부 플레이테스터에게 실행 파일을 전달할 수 있다.
- 첫인상에서 프로토타입 냄새가 크게 줄어든다.
- 1런 완주 경험이 매끄럽다.

## 현재 위치

- 현재 위치는 `M1 완료 + M2 진행 중`
- 이미 된 것:
  - 챔피언 선택
  - 시작 배치
  - 3전 원정
  - 보상
  - 경로 선택
  - 경로 이벤트 2차
  - 챔피언 고유성 2차
  - 챔피언별 스킬 차별화 2차
  - 보스전 1차
  - 노드 분기 1차
  - 노드 후속 이벤트
  - 보스전 2차
  - 보스 패턴 2종과 결전 스테이지 변주
  - 런 종료 요약
  - 적 의도 4차
  - 적 의도 3차 1단계
  - 지형
  - 엘리트
  - 경로 목표 1차
  - AI 목표 경쟁 1차
  - 목표 실패 페널티 1차
- 아직 큰 구멍:
  - 노드 전용 컷신/연출
  - 콘텐츠 볼륨
  - 출시급 비주얼

## 다음 3개 작업

1. 런 기록 저장/최고 기록 비교와 결산 화면 확장
2. 노드 전용 컷신/전투 시작 연출 추가
3. 보스 패턴 시스템 위에 보스별 추가 특수 규칙/연출 확장
