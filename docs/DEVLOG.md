# Devlog

## 2026-03-10

### 보스전 1차와 엘리트 특수 규칙

- `native_tactics` 3전 결전에 보스 개체를 추가하고, 2전 이후 엘리트 적에는 `철벽`, `맹추격`, `비전 폭주` 특성을 부여
- 보스는 일반 엘리트와 분리해 더 큰 체력/보호막/이동력/사거리 보정을 받고, 체력 절반 이하에서 한 번 `결전 각성`을 발동
- 결전 각성은 보호막 18, 속도 +2, 이동력 +1, 공격 강화, 특수기 쿨다운 감소를 즉시 적용해 마지막 전투를 실제로 다르게 느끼게 만듦
- 배치 화면, 전투 화면, 우측 현황 패널, 최근 로그에 보스/엘리트 특성을 직접 노출해 플레이어가 현재 위협 등급을 바로 읽을 수 있게 조정
- 보스 지정, 엘리트 특성, 보스 페이즈 전환을 테스트로 고정하고 결전 전장 스크린샷도 별도로 확인

### 챔피언 고유성 2차

- `native_tactics`의 전술용 `특수기`를 챔피언별 명시 매핑으로 고정해 다리우스/제드처럼 방어기가 잘못 선택되던 문제를 정리
- 아리는 군중 제어형, 징크스/럭스/이즈리얼은 궁극기형, 레오나/브라움은 전장 제압형처럼 챔피언별 전술 역할이 더 분명하게 드러나도록 재정의
- 이동 거리, 고립, 기절 연계, 다중 타겟 수, 지형 생성, 처치 보상 같은 조건을 활용해 패시브 2차 트리거를 추가
- 럭스의 룬 생성, 레오나의 광역 기절 보강, 브랜드의 화염 지대 생성, 이즈리얼의 특수기 쿨다운 환급처럼 전술 맵 상호작용을 늘림
- `special` 선택 규칙이 전체 엔진에 영향을 주므로 `py_compile`, `tactics:test`, `tactics:smoke`, `tactics:capture`를 다시 전부 통과시켜 회귀를 고정

### 경로 이벤트 2차와 목표 실패 페널티

- 각 경로 카드에 무작위 `전투 이벤트`를 붙여 다음 전투 한정 추가 스탯 변화를 부여
- 이벤트는 선택 화면 장식이 아니라 실제 다음 전투 스탯에 적용되도록 엔진 보정에 연결
- 맵 목표를 실패한 채 승리하면 다음 전투용 페널티를 예약하고, 경로 화면과 배치 화면까지 그 상태를 이어서 보여 주도록 확장
- `보상 -> 경로 -> 배치 -> 다음 전투` 흐름에서 이벤트와 페널티가 실제로 반영되는 회귀 테스트를 추가
- 경로 화면 카드 높이와 문구 밀도를 조정해 이벤트/실패 정보가 버튼과 겹치지 않도록 정리

### 적 의도 3차 1단계와 목표 경쟁 AI

- 적 의도 패널에 `적 연속 턴` 요약을 추가해 총 예상 피해와 위협 칸 수를 함께 보여 주도록 확장
- 현재 적 행동과 다음 적 경고에 광역 타겟 수와 목표 압박 여부를 같이 표기
- 전장 오버레이도 현재 위협, 다음 위협, 추가 연속 턴 위협, 목표 압박 타일을 구분해 렌더링
- 레드 AI가 점령 목표 타일과 지형형 목표 지점을 실제로 두고 싸우도록 목표 경쟁 점수 함수를 추가
- 전술 목표 상태와 엔진의 목표 타일을 동기화하는 별도 헬퍼를 두어 전투 중 달성/실패 후에도 AI가 즉시 반응하도록 정리
- `적 연속 턴 요약`, `목표 타일 경쟁 이동`, `전투 시작 시 목표 동기화`를 테스트로 고정
