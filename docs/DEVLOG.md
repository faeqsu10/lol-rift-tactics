### 전투 HUD 패널 2차 정리

- 전투 좌측 패널은 `상태 카드`, `유닛 정보 카드`, `패시브 카드`, `적 의도 카드`, `조작 카드`로 시각 규칙을 통일해 정보 구획이 한눈에 보이도록 재정리
- 전투 우측 패널에 `턴 순서 스트립`을 추가하고, 팀 로스터 행에 체력바를 붙여 각 유닛 상태를 숫자와 막대로 동시에 읽을 수 있게 보강
- 결전 전투에서는 상단에 결전 요약 카드를 별도로 두어 보스 패턴/목표/특수 규칙 정보를 팀 로스터와 겹치지 않게 분리
- 하단 액션 바 우측 정보를 별도 카드로 묶어 `현재 턴`, `이동/행동 상태`, `전술 목표`가 버튼 영역과 충돌하지 않도록 조정

### 전투 HUD 패널 3차 겹침 수정

- 좌측 HUD를 고정 좌표 누적 방식에서 `상태 -> 메타 -> 유닛 -> 패시브 -> 적 의도 -> 조작` 순서의 `bottom + 여백` 레이아웃으로 바꿔, `조작` 카드가 하단 액션 바와 겹치던 문제를 제거
- 유닛 정보 카드/하단 정보 카드/우측 로스터 행에 텍스트 자동 축약(ellipsis)을 도입해 긴 스킬명·상태 문구가 카드 경계를 넘지 않게 정리
- 하단 액션 버튼은 라벨 길이에 따라 `UI 폰트`와 `small 폰트`를 자동 선택하도록 조정해 긴 특수기 이름에서도 버튼 내부 가독성을 유지
- 전투 시작 컷인 유무 2종 전장 캡처와 `tactics:test`, `tactics:smoke`, `tactics:capture`를 다시 통과시켜 레이아웃 회귀를 고정

### 선택/경로/결산 화면 UI 정리

- `native_tactics` 상단 헤더를 `제목`, `현재 단계`, `행동 버튼`으로 분리해 긴 한글 제목과 상태 문구가 겹치지 않게 재구성
- 선택 화면은 `출전 라인업`, `후보 로스터`, `원정 교리`, `예상 적 조합`으로 구획을 다시 나누고, 카드마다 핵심 한 줄만 남겨 텍스트 밀도를 줄임
- 경로 선택 화면은 좌측 `전투 요약/핵심 수치/다음 전투 프리뷰`, 우측 `3안 경로 카드`로 재배치하고, 경로 카드 안에서는 목표·노드·이벤트·보상·위험만 남겨 읽기 흐름을 단순화
- 결산 화면은 좌측 하단 정보를 `기록·해금·다음 추천` 한 카드로 합치고, 즉시 재도전 버튼 영역을 별도로 빼서 하단 겹침을 없앰
- 선택/경로/결산 세 화면은 헤드리스 캡처와 `tactics:test`, `tactics:smoke`, `tactics:capture`로 다시 검증

### 저장 기록 기반 원정 교리와 경로 재추첨

- `native_tactics` 저장 기록을 단순 결산 비교에서 끝내지 않고, `보급 교범`, `기동 교범`, `정찰 네트워크` 3종 `원정 교리`로 연결
- 교리는 `기록 수`와 `완주 수`로 해금되며, 선택 화면에서 런 시작 전에 1개를 직접 고르게 만들어 다음 런 동기를 눈에 보이게 정리
- `보급 교범`과 `기동 교범`은 시작 강화 1개를 미리 부여하고, `정찰 네트워크`는 경로 선택 화면에서 `경로 재추첨` 1회를 제공
- 결산 화면에는 저장 기록 비교와 함께 `신규 교리 해금` 문구를 노출하고, 선택 화면/경로 화면/런 빌드 요약에도 같은 교리 정보가 보이도록 연결
- 기록 해금, 시작 보너스 적용, 경로 재추첨 소비, 신규 해금 문구는 테스트와 헤드리스 캡처로 다시 검증

### 보스별 추가 특수 규칙과 결전 위험 타일 예고

- `native_tactics` 결전 보스 각성에 패턴별 추가 효과를 얹어, `전장 군주`는 `전선 붕괴`, `룬 폭주`는 `룬 폭풍`으로 즉시 압박 피해를 주도록 확장
- 이 추가 피해는 일반 스킬 타격과 분리해 처리해, 챔피언 처치/다중 타겟 패시브가 보스 각성 부가 효과에 잘못 반응하지 않게 정리
- 엔진에 보스 패턴별 압박 타일 계산을 분리하고, 전장에는 `예고/활성` 상태가 다른 색으로 표시되도록 결전 오버레이를 추가
- 결전 우측 패널과 중앙 컷인에도 `각성 규칙` 문구를 노출해, 마지막 전투 설명과 실제 위협 구역이 같은 정보를 가리키도록 맞춤
- `전장 군주`와 `룬 폭주`의 추가 압박 피해, 압박 타일 계산, 결전 컷인 문구는 테스트와 결전 전장 헤드리스 캡처로 다시 검증

- `전장 군주`와 `룬 폭주` 각성, 결전 목표, 결전 전용 장애물 레이아웃은 테스트로 고정하고 `tactics:test`, `tactics:smoke`, `tactics:capture`를 다시 통과시킴

- 배치 이탈 복귀와 노드 요약 문자열은 회귀 테스트로 고정하고 전체 `tactics:test`, `tactics:smoke`를 다시 통과시킴

### 노드 후속 이벤트와 브리핑 연동

- `휴식 거점`, `변수 균열`, `정예 수배`가 단순 고정 설명이 아니라, 각 노드마다 한 번 더 굴러가는 `후속 이벤트`를 갖도록 확장
- 후속 이벤트는 다음 전투 한정 추가 스탯 보정을 제공하고, 경로 카드/다음 전투 정보/배치 브리핑/전투 HUD/전투 로그가 모두 같은 결과를 보도록 연결
- 휴식 노드는 회복/재집결, 변수 노드는 폭주/흡수, 정예 노드는 현상금/추적처럼 같은 노드 안에서도 다른 전투 감각이 나도록 구성
- 노드 후속 이벤트는 별도 계산 경로가 아니라 기존 스테이지 보정 합산에 들어가게 붙여 프리뷰와 실적용 차이가 없도록 유지
- 경로 생성 시 후속 이벤트가 항상 같이 붙는지, 후속 이벤트 수치가 실제 전투 이동력/속도 등에 반영되는지 테스트로 고정하고 경로 화면 캡처로 카드 밀도도 다시 확인

### 런 종료 요약과 재도전 유도 강화

- `native_tactics` 최종 승리와 패배가 더 이상 전투 오버레이에 머물지 않고, 별도 `원정 결산` 화면으로 넘어가도록 재구성
- 결산 화면에는 출전 조합, 누적 라운드/피해/처치, 주력 강화, 마지막 목표/노드/이벤트, 스테이지별 전투 타임라인을 한 번에 노출
- 결산 화면 하단에 `같은 조합 새 원정`, `캐릭터 다시 선택` 버튼과 단축키를 같이 두어 한 판 뒤 다음 행동이 바로 보이도록 정리
- 전투 기록은 런 단위 히스토리로 누적하고, 중간 승리는 기존 보상/경로 흐름으로 보내면서 최종 승리와 패배만 결산 화면으로 분기하도록 분리
- `중간 승리 -> 보상`, `최종 승리 -> 결산`, `패배 -> 결산`, `결산 -> 새 런` 상태 전환을 회귀 테스트와 헤드리스 요약 캡처로 고정

### 런 노드 분기 1차

- `native_tactics` 경로 선택 카드에 `휴식 거점`, `변수 균열`, `정예 수배` 3종 런 노드를 추가해 경로 선택이 실제 메타 선택으로 느껴지도록 확장
- 휴식 거점은 다음 전투 아군 체력/보호막을 올리고 예약된 실패 페널티를 정리하도록 연결
- 변수 균열은 경로 이벤트의 전투 보정과 목표 실패 페널티를 함께 증폭해 고위험 고효율 선택지가 되도록 구성
- 정예 수배는 다음 전투 정예 수를 늘리는 대신, 승리 시 추가 런 강화 1개를 자동 지급하도록 연결
- 카드 프리뷰, 다음 전투 정보, 배치 화면, 전투 로그, HUD가 모두 같은 노드/이벤트 계산값을 보도록 정리해 선택 화면과 실제 전투 적용 차이를 없앰
- 런 노드 생성, 휴식 노드 페널티 해제, 변수 노드 보정 증폭, 정예 노드 추가 보상은 회귀 테스트로 고정

### 보스전 2차와 결전 목표

- 결전 스테이지는 이제 일반 경로 목표 대신 `결전 봉쇄`, `핵심 차단` 같은 보스 전용 목표를 사용
- 보스 역할에 따라 목표 종류와 점유 칸 수를 달리 잡아 마지막 전투가 단순 전멸전이 아니라 `보스 각성 약화`를 노리는 전술전으로 바뀌도록 조정
- 결전 목표를 성공하면 보스의 결전 각성 보호막/속도를 약화하고, 실패하면 각성을 더 증폭하도록 연결
- 보스가 체력 절반 이하에서 각성하면 주변 칸에 `결전 파동`이 퍼져 화염 지대를 만들고, 상단 결전 배너와 우측 패널 결전 상태가 함께 갱신되도록 추가
- 경로 화면과 배치 화면도 결전 전용 목표를 미리 보여 주도록 바꿔, 최종전 진입 전 설명과 실제 전투 규칙이 어긋나지 않게 정리
- 결전 목표 생성, 결전 파동 지형, 성공/실패에 따른 보스 약화/강화는 테스트로 고정

### 적 의도 4차와 위험도 계산

- 적 의도 계산에 유닛별 예상 피해 맵을 추가해 `누가 얼마나 맞는지`를 연속 턴 기준으로 합산하도록 확장
- 적 연속 턴 요약은 이제 `위험 낮음/보통/높음/치명` 등급과 `연쇄 집중`, `처치각`, `동시 압박` 문구를 같이 표시
- 동일 타겟에 포커스가 겹칠 때는 전장 위에 붉은 포커스 테두리를 추가해 현재 가장 위험한 아군을 즉시 읽을 수 있게 함
- 적 의도 패널 높이와 조작 패널 밀도를 재조정해 위험도 한 줄이 추가돼도 전투 HUD가 무너지지 않도록 정리
- 연쇄 집중 대상, 치명 위험 타겟 수, 포커스 대상 계산을 테스트로 고정하고 전투 화면 캡처로 패널 레이아웃을 다시 확인

### 보스전 1차와 엘리트 특수 규칙

- `native_tactics` 3전 결전에 보스 개체를 추가하고, 2전 이후 엘리트 적에는 `철벽`, `맹추격`, `비전 폭주` 특성을 부여
- 보스는 일반 엘리트와 분리해 더 큰 체력/보호막/이동력/사거리 보정을 받고, 체력 절반 이하에서 한 번 `결전 각성`을 발동
- 결전 각성은 보호막 18, 속도 +2, 이동력 +1, 공격 강화, 특수기 쿨다운 감소를 즉시 적용해 마지막 전투를 실제로 다르게 느끼게 만듦
- 배치 화면, 전투 화면, 우측 현황 패널, 최근 로그에 보스/엘리트 특성을 직접 노출해 플레이어가 현재 위협 등급을 바로 읽을 수 있게 조정
- 보스 지정, 엘리트 특성, 보스 페이즈 전환을 테스트로 고정하고 결전 전장 스크린샷도 별도로 확인

### 챔피언 고유성 2차

- `native_tactics`의 전술용 `특수기`를 챔피언별 명시 매핑으로 고정해 다리우스/제드처럼 방어기가 잘못 선택되던 문제를 정리
- 아리는 군중 제어형, 징크스/럭스/이즈리얼은 궁극기형, 레오나/브라움은 전장 제압형처럼 챔피언별 전술 역할이 더 분명하게 드러나도록 재정의
- 이동 거리, 고립, 기절 연계, 다중 타겟 수, 지형 생성, 처치 보상 같은 조건을 활용해 패시브 2차 트리거를 추가
- 럭스의 룬 생성, 레오나의 광역 기절 보강, 브랜드의 화염 지대 생성, 이즈리얼의 특수기 쿨다운 환급처럼 전술 맵 상호작용을 늘림
- `special` 선택 규칙이 전체 엔진에 영향을 주므로 `py_compile`, `tactics:test`, `tactics:smoke`, `tactics:capture`를 다시 전부 통과시켜 회귀를 고정

### 경로 이벤트 2차와 목표 실패 페널티

- 각 경로 카드에 무작위 `전투 이벤트`를 붙여 다음 전투 한정 추가 스탯 변화를 부여
- 이벤트는 선택 화면 장식이 아니라 실제 다음 전투 스탯에 적용되도록 엔진 보정에 연결
- 맵 목표를 실패한 채 승리하면 다음 전투용 페널티를 예약하고, 경로 화면과 배치 화면까지 그 상태를 이어서 보여 주도록 확장
- `보상 -> 경로 -> 배치 -> 다음 전투` 흐름에서 이벤트와 페널티가 실제로 반영되는 회귀 테스트를 추가
- 경로 화면 카드 높이와 문구 밀도를 조정해 이벤트/실패 정보가 버튼과 겹치지 않도록 정리

### 적 의도 3차 1단계와 목표 경쟁 AI

- 적 의도 패널에 `적 연속 턴` 요약을 추가해 총 예상 피해와 위협 칸 수를 함께 보여 주도록 확장
- 현재 적 행동과 다음 적 경고에 광역 타겟 수와 목표 압박 여부를 같이 표기
- 전장 오버레이도 현재 위협, 다음 위협, 추가 연속 턴 위협, 목표 압박 타일을 구분해 렌더링
- 레드 AI가 점령 목표 타일과 지형형 목표 지점을 실제로 두고 싸우도록 목표 경쟁 점수 함수를 추가
- 전술 목표 상태와 엔진의 목표 타일을 동기화하는 별도 헬퍼를 두어 전투 중 달성/실패 후에도 AI가 즉시 반응하도록 정리
- `적 연속 턴 요약`, `목표 타일 경쟁 이동`, `전투 시작 시 목표 동기화`를 테스트로 고정
