- 헤드리스 환경에서 `선택 -> 배치 -> 전투` 상태 전환을 직접 검증

### 전투 종료 복귀 UX 개선

- `native_tactics` 승패 오버레이에 `캐릭터 다시 선택`, `같은 조합 재대결` 버튼 추가
- 전투가 끝난 뒤 키를 외워야만 빠져나오는 구조 대신 화면에서 직접 다음 행동을 고를 수 있게 변경
- `Enter`와 `ESC`는 선택 화면 복귀, `R`은 즉시 재대결로 정리

### 전술 원정 루프 추가

- `native_tactics`를 단발 전투에서 `정찰전 -> 교전 -> 결전` 3전 원정 구조로 확장
- 중간 승리 뒤 전역 강화 보상 3종 중 하나를 고르게 하고, 선택 후 다음 전투 배치 단계로 이동하도록 추가
- 보상 화면에서 다음 적 조합을 미리 보여 줘서 선택의 의미를 강화
- 런 종료 시 `같은 조합 새 런` 또는 `캐릭터 다시 선택`으로 자연스럽게 재진입 가능하도록 정리
- UX/리텐션 근거 문서를 `docs/TACTICS_UX_RESEARCH.md`로 추가
