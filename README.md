# League of Legends: Rift Tactics

> 3명을 골라라. 배치하라. 살아남아라.

LoL 세계관 기반 팬메이드 턴제 전술 게임.
Pygame으로 만든 네이티브 클라이언트 — 브라우저 없이 바로 플레이.

## 게임 흐름

```
챔피언 선택 (11인 중 3인)
    -> 시작 배치
        -> 정찰전
            -> 보상 선택 + 경로 분기
                -> 엘리트전
                    -> 보상 선택 + 경로 분기
                        -> 보스 결전
                            -> 원정 결산
```

3전 원정을 완주하면 결산. 기록을 쌓으면 **원정 교리**가 해금되어 다음 런이 달라집니다.

## 뭐가 있나

**전투**
- 그리드 위 이동 + 스킬 시전 턴제 전투
- 챔피언마다 고유 패시브 + 전술 특수기
- 수풀/룬/화염 지형이 전투를 바꿈
- 적 AI가 지형, 목표, 위협을 계산해서 움직임

**원정**
- 매 승리 후 보상 1개 + 3갈래 경로 선택
- 경로마다 리스크/보상 + 맵 목표 + 전투 이벤트
- 휴식 거점 / 변수 균열 / 정예 수배 노드 분기
- 런 기록 저장, 최고 기록 비교, 교리 해금

**보스전**
- 3전째는 보스 결전 — 전장 군주 / 룬 폭주 / 서리 공성 중 하나
- 보스 체력 절반 이하 시 **결전 각성** 발동
- 결전 전용 지형 + 위험 타일 + 보스 패턴 변주

**연출**
- 전술 스탠디 (초상화 + 팀색 배너 + 역할 장비)
- 공격 돌진 / 피격 흔들림 / 사망 페이드 / 승리 포즈
- 스테이지별 맵 테마 (초원 / 룬 / 결전)
- 전투 시작 컷인 + 모티프 배지 + 테마 사운드

## 바로 플레이

```bash
npm run native:setup   # 처음 한 번
npm run tactics:play   # 게임 시작
```

## 조작

| 입력 | 동작 |
|------|------|
| 클릭 | 챔피언 선택, 이동, 스킬 타겟 |
| `1` `2` `3` | 스킬 선택 |
| `R` | 전투 재시작 |
| `ESC` | 뒤로가기 / 종료 |
| `H` / `F1` | 도움말 |
| `F10` | 설정 (볼륨, 전투 속도, 난이도) |

## 챔피언 풀 (22명)

| 블루 (11) | 레드 (11) |
|-----------|-----------|
| 가렌, 아리, 징크스, 럭스, 바이 | 다리우스, 애니, 케이틀린, 모르가나, 야스오 |
| 이즈리얼, 레오나, 애쉬, 리븐, 아칼리, 브라움 | 제드, 리산드라, 카타리나, 브랜드, 오리아나, 세트 |

## 테스트 & 빌드

```bash
npm run tactics:test      # 전술 엔진 103개 테스트
npm run tactics:smoke     # 헤드리스 스모크
npm run tactics:capture   # 스크린샷 캡처
npm run native:package    # Linux 실행 파일 빌드
```

Windows `.exe`는 GitHub Actions 아티팩트에서 다운로드.

## 기술 스택

- **엔진**: Pygame-CE (Python)
- **렌더링**: 1920x1080 디자인 해상도 -> 윈도우 스케일링
- **오디오**: 챔피언별 오리지널 신스 모티프 (Riot 음원 미사용)
- **폰트**: 한국어 번들 (NanumSquareRoundEB)
- **패키징**: PyInstaller (Linux/Windows)
