Optional tactical standee cutout assets live here.

Expected format:
- PNG with alpha transparency
- Prefer vertical cutout / bust or half-body art
- Use the same filenames as `assets/champions/`

Examples:
- `Garen.png`
- `Ahri.png`
- `Jinx.png`

If a file is missing here, `native_tactics` falls back to a synthesized cutout built from
the square portrait art in `assets/champions/`.
